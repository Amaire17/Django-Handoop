WHAT IS BIG DATA?

Part of the horizontal scale of growing data feed by the response of integrative programming and technologies whether the data is from structured, unstructured, semi-structured.

WHAT IS HADOOP?

Hadoop is a Java-based open source that distributed storage and processing of very large data sets on commodity hardware clusters. It provides massive storage for any kind of data, enormous processing power and the ability to handle big data and organize them more quickly and efficiently.

WHAT ARE THE COMPONENTS OF HADOOP?

There are 3 components of Hadoop which are Hadoop HDFS, Hadoop MapReduce, Hadoop YARN.

Hadoop Distributed File System

Is a storage model it allows data to be stored in files. There are two types of nodes in HDFS: NameNode and DataNode. The commodity servers where the data is really kept are known as DataNodes. The NameNode, on the other hand, holds metadata that describes the data contained in the various nodes.


Yet Another Resource Negotiator (YARN)

A resource-management platform that is responsible for managing compute resources in clusters and using them for scheduling of users' applications.

MAP REDUCE

It is very important and major components in Hadoop where it divide huge data set into smaller pieces. MapReduce is a programming model that is used for efficient processing in parallel over large data-sets in a distributed manner.
